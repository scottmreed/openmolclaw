Chat

Ask about a molecule or a SMILES string. The assistant uses local RDKit tools (validate, render, descriptors, substructure/SMARTS search, functional groups, stereochemistry) and answers with the model you selected above. Nothing is sent to a ChemIllusion server.

    Zero Data Retention (ZDR) is OpenRouter's routing option that restricts a request to endpoints that have committed not to retain or train on it. ZDR Mode only affects requests this local app sends to OpenRouter — it does not change your OpenRouter account settings, OS logs, browser extensions, proxies, custom providers, or local workspace files. Private Structure Mode combines ZDR (when a hosted provider is used), blocked external molecule lookups, and a memory-only workspace into one toggle; see /docs for the exact claim it earns.
